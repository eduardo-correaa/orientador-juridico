# Orientador Jurídico

Plataforma web desenvolvida no Hackathon Juris 2025 (3º lugar), com o objetivo de facilitar o acesso à informação jurídica para usuários leigos, utilizando inteligência artificial para simplificação de respostas.

## Funcionalidades

- Cadastro e login de usuários
- Chatbot jurídico com IA
- Busca pública de processos
- Sistema de planos (básico, intermediário e premium)
- Página de recomendação de advogados (LexClips)

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Node.js
- Express
- MySQL
- Bcrypt (criptografia de senha)
